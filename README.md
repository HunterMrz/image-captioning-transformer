# Dual-CNN Region-Level Feature Fusion for Image Captioning Using Transformer Decoder

## Overview

This project focuses on image captioning by combining Computer Vision and Natural Language Processing. The approach uses CNN-based visual feature extraction with a Transformer decoder architecture to generate captions from images.

## Research Areas

## Technologies Used

- Python
- Deep Learning

- Transformer Decoder
- Jupyter Notebook / Google Colab
- Computer Vision
- NLP

## Project Description

The project developed an image captioning approach using Dual-CNN region-level feature fusion. Visual features are extracted from images and used by a Transformer decoder to generate natural language captions.

This project is related to AI, machine learning, visual computing, and data-driven intelligent systems.
